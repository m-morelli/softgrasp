lspb

Linear segment with parabolic blend

[s,sd,sdd] = lspb(s0, sf, T) as above but specifies the trajectory in terms of the length of the time vector T. [s,sd,sdd] = lspb(s0, sf, T, v) as above but specifies the velocity of the linear segment which is normally computed automatically and a time vector.

Notes

  • in all cases if no output arguments are specified S, SD, and SDD are plotted against time.
  • for some values of V no solution is possible and an error is flagged.
