M-File Help: plot_point View code for plot_point

plot_point

point features

'textcolor', colspec Specify color of text 'textsize', size Specify size of text 'bold' Text in bold font. 'printf', {fmt, data} Label points according to printf format string and corresponding element of data 'sequence' Label points sequentially Additional options are passed through to PLOT for creating the marker.
