trplot2

Plot a planar transformation

'color', c The color to draw the axes, Matlab colorspec. 'axes' Set dimensions of the Matlab axes 'frame', f The name which appears on the axis labels and the frame itself 'text_opts', opt A cell array of Matlab text properties 'arrow' Use arrows rather than line segments for the axes 'width', w Width of arrow tips 'handle', h Draw in the Matlab axes specified by h
