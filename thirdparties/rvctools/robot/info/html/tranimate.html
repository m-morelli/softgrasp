tranimate

Animate a coordinate frame

tranimate(p, options) animates a coordinate frame moving from the identity pose to the pose p represented by any of the types listed above. 'fps', fps Number of frames per second to display (default 10) 'nsteps', n The number of steps along the path (default 50)
